CIMI_Clinical.InformationEntry

Overview of the InformationEntry Package and Related Inheritance Hierarchy.

Package for base InformationEntry class (green in diagram).

The InformationEntry class is the core information entry class in CIMI and represents an entry in a record. It is an abstract class that has several concrete subclasses.
